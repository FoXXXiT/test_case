## INSTALL AND RUN

```bash
cp .env.example .env
```

```bash
composer install
```

```bash
./vendor/bin/sail up -d
```

```bash
./vendor/bin/sail php artisan migrate
```

```bash
./vendor/bin/sail php artisan db:seed
```

<strong>Then you can access resource through</strong>
[http://localhost](http://localhost/)
